--->Como compilar:
	 Desde la terminal ir a la dirección -/ICC/practica04/src$
	 y ejecutar :
	  -/ICC/practica04/src$ javac main/Main.java

	  *Con esto se compila todos los archivos que necesita el Main.
	  
----> Como ejecutar:

      Desde la terminal ir a la dirección  -/ICC/practica04/src$

      y ejecutar:
      	 -/ICC/practica04/src$ java main/Main > NombreQueQuieras.html

	 *Con esto la ejecución del programa no se ve en la terminal si no que esta lo dereccióna a un archivo llamado NombreQueQuieras.html el cual podemos encontrar en la carpeta src y lo podemos abrir desde cualquier navegador.
