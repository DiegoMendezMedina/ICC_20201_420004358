--->Como compilar y ejecutar:
	 En la direccion de este archivo se encuentra un documento de extension .sh el cual al ejecutarlo compila y corre el programa.

	 Si quieres usar el script recuerda que debes de darle permisos al archivo con chmod. Una ves dado los permisos desde la direccion de este archivo ejecutar ./script.sh

Otra opcion para compilar y ejecutar:

--->Como compilar:
	 Desde la terminal ir a la dirección -/ICC/practica07

	 y ejecutar :
  	   javac chess/Game.java  *Con esto se compila el programa y todos los archivos que vaya a necesitar.
	  
----> Como ejecutar:

	 Desde la terminal ir a la dirección -/ICC/practica07

	 y ejecutar:
	 
	java chess/Game 	 *Con esto el programa se ejcuta
