     Desde la direcciòn ICC/pracitca08/
     *dar permisos al script compile.sh
     	  sudo chmod 775 compile.sh
     *ejecutar el script
     	  ./compile.sh
	  
Para ejecutar
Desde la direcciòn ICC/pracitca08/
     *dar permisos al script run.sh
     	  sudo chmod 775 run.sh
     *ejecutar el script
     	  ./run.sh
