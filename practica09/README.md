Para compilar y ejecutar la practica09
     Desde la direcciòn ICC/pracitca09/
     *dar permisos al script CompileRun.sh
     	  sudo chmod 775 CompileRun.sh
     *ejecutar el script
     	  ./compileRun.sh

El script compila el Reader y Draw y ejecuta el archivo Draw el cual nos genera una ventana en processing

Tambien se puede compilar y ejecutar llendo a ICC/practica09/src y ejecutando:

	javac Reader.java --Compila el Reader
	javac Draw.java   -- Compila Draw
	java Draw        -- Ejecuta Draw
